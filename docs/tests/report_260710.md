# 测试报告

## 设置页面
### 安全设置
- 自动锁定时间的下拉展开的默认选项和显示的值不一致；
- 密码锁解锁界面的删除按键改成回退，图标改成回退样式的；
- 密码锁启用时的录入密码，要么用数字键盘要么弹出系统的数字键盘，现在弹出的是英文键盘，这里要和解锁界面保持一样的用户体验；
- 修复密码的输入键盘保持和启用密码锁的输入密码用户体验一致；
- 重置密码界面，我点【取消】后，界面上的密码锁开关直接显示了关闭，实际上我刷新下还是打开的；
