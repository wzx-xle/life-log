## 为什么

LifeLog 解决了一个隐私优先、纯本地的线下服务场所体验记录需求。目前没有一款轻量级 PWA 能够结合地图浏览、详细评价追踪和完整离线能力，且无需账号或云存储。本 MVP 使用户能够构建一个完全由自己掌控的个人消费日记。

## 变更内容

- 使用 Vue 3 + TypeScript + Vite 搭建移动优先的 PWA 项目
- 实现 4 位数字密码锁：SHA-256 哈希、自动锁定、暴力破解防护
- 使用高德地图 JSAPI 2.0 构建地图驱动的主页，含店铺标记、搜索和定位
- 实现店铺完整 CRUD：分类、照片、标签、地图选点
- 实现两种评价模式：快速模式（单评分 + 简短笔记）和详细模式（多维度评分、消费项目、会再光顾）
- 构建评价浏览列表：按时间分组、分类筛选、排序、关键词搜索
- 构建统计面板：图表（趋势、分类占比、评分分布）和摘要卡片
- 实现数据导出（JSON 下载）、导入（合并/覆盖）、清除全部数据
- 添加 PWA 支持：manifest、Service Worker 预缓存 + 高德瓦片运行时缓存
- 配置设置页面：密码锁、自动锁定时间、默认评价模式、地图缩放、关于信息

## 能力

### 新增能力
- `password-protection`：4 位应用密码锁，支持密码创建/修改/重置、自动锁定计时器、暴力破解锁定（5 次错误 → 30 秒）
- `map-view`：基于高德地图的主视图，含店铺标记（按分类着色）、POI 搜索、用户定位、长按添加、地图选点器
- `place-management`：店铺的增删改查，含名称、分类（6 种预设 + 自定义）、地址、经纬度、电话、营业时间、标签、照片（最多 9 张，自动压缩）
- `review-records`：快速模式（评分 + 短笔记）和详细模式（4 维度评分、消费项目、标签、会再光顾）；编辑/删除
- `review-browsing`：按时间分组的列表视图，支持分类筛选、排序（时间/评分/距离）、关键词搜索
- `place-detail`：店铺详情页，含照片轮播、综合评分、信息、操作（导航、拨打电话）、关联评价列表、编辑/删除
- `stats-dashboard`：摘要卡片、月度消费趋势折线图、分类饼图、评分分布柱状图、常去店铺 Top5、足迹地图入口
- `data-management`：JSON 导出（含版本号）、导入（合并/覆盖模式）、清除全部数据（含确认）
- `settings`：安全设置（密码开关、修改、自动锁定、重置）、偏好设置（默认评价模式、地图缩放）、关于信息

## 影响

- 新项目脚手架：Vue 3 + Vite + TypeScript + Vant 4 + Pinia + Vue Router + Dexie.js
- 新增依赖：高德地图 JSAPI 2.0、ECharts 5、Compressor.js、vite-plugin-pwa、postcss-px-to-viewport
- 数据完全存储在 IndexedDB（通过 Dexie.js，places + reviews 两张表）和 localStorage（设置项）
- 无服务端变更——纯客户端、离线优先的 PWA
