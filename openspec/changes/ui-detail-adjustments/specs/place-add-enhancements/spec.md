## ADDED Requirements

### Requirement: 新增店铺页固定标题
新增店铺页的标题栏 SHALL 固定在视口顶部，不随内容滚动。

#### Scenario: 新增店铺页标题固定
- **WHEN** 用户在新店铺添加页向下滚动填写表单
- **THEN** 标题栏保持在视口顶部可见

### Requirement: 返回按钮仅保留箭头
新增店铺页的标题栏返回按钮 SHALL 仅显示箭头图标，不显示「返回」文字。

#### Scenario: 返回按钮无文字
- **WHEN** 用户打开新增店铺页
- **THEN** 标题栏左侧仅显示箭头图标，不显示「返回」文字

### Requirement: 添加店铺表单底部取消按钮
新增店铺页 SHALL 在提交按钮下方显示取消按钮。

#### Scenario: 取消按钮可见
- **WHEN** 用户在新增店铺页
- **THEN** 「添加店铺」按钮下方显示「取消」按钮

#### Scenario: 点击取消返回
- **WHEN** 用户点击「取消」按钮
- **THEN** 导航返回上一页

### Requirement: 分类选项末尾新增按钮
分类选择网格 SHALL 在末端显示一个仅包含 + 号的按钮，点击可新增分类。

#### Scenario: + 按钮可见
- **WHEN** 用户在新增/编辑店铺页的分类区域
- **THEN** 分类网格末尾显示一个仅包含 + 号的按钮

#### Scenario: 点击 + 弹出新增分类框
- **WHEN** 用户点击分类网格末尾的 + 按钮
- **THEN** 弹出新增分类表单（名称 + 颜色 + 确认/取消）

#### Scenario: 新增分类后刷新列表
- **WHEN** 用户在新增分类框中完成创建
- **THEN** 分类网格自动刷新，新分类出现并可选
